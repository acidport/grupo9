Información del OVI

Este es un Objeto Virtual de Información O.V.I. del curso de Diseño de Sitios Web en el cual encontrara información básica acerca de cómo realizar y administrar una base de datos en SQL.

Entre los conceptos básicos que se miraran en este OVI están:

  • LDD - Lenguaje de Definición de Datos (DDL inglés).
  • LCD - Lenguaje de Control de Datos (DCL inglés)
  • LMD - Lenguaje de Manipulación de Datos (DML inglés)
  • Estandarización y Ejemplo de una Base de Datos.

Objetivo General

Explicar los pasos en la Implementación de un Modelo Relacional sobre un SGBD (SQL Server), así mismo es importante resaltar el uso de LDD, LCD y LMD.

Objetivos Específicos

  • Manejar un pequeño estándar para la creación y administración de las bases de datos.
  • Brindar información básica pero necesaria para el desarrollo e implementación de un sistema de bases de datos.
  • Hacer una base de datos referencial pequeña para la mejor comprensión de lo expuesto.
