LCD

Que es el LCD en bases de datos.
        Es un lenguaje proporcionado por el Sistema de Gestión de Base de Datos que 
        incluye una serie de comandos SQL que permiten al administrador controlar el 
        acceso a los datos contenidos en la Base de Datos.
        Sistemas gestores de base de datos que utilizan SQL:
        SQL Server
        Oracle
        MySql
        PostgreSQL
        DDL:
        - Es la parte del SQL dedicada a la definición de la base de datos.
        - Permite crear índices, definir reglas de validación de datos, relaciones 
          entre tablas, etc.
        DDL (Lenguaje de definición de datos)
        DML (Lenguaje de manipulación de datos)
        DCL (Lenguaje de control de datos)
        Lenguaje estructurado de consulta (Structured Query Language)
        Lenguaje utilizado para definir, controlar y accerder a los datos almacenados 
        en una BD relacional.

        Es un lenguaje universal que se emplea en cualquier sistema gestor de datos 
        relacional.
        En SQL Server la versión de SQL que se utiliza se llama TRANSACT - SQL.
        DML:
        Se compone de las instrucciones para el manejo de los datos es decir para 
        insertar, modificar, eliminar, y seleccionar datos
        DCL:
        Se componen de instrucciones que permiten ejercer un control sobre los datos. 
        Es responsable de la gestión de transacciones (COMMIT/ROLLBACK).
