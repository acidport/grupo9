DML

        El lenguaje de manipulación de datos (DML) es un vocabulario utilizado 
        para recuperar y trabajar con datos en SQL. Existen declaraciones las cuales 
        se podrán realizar con cada una de ellas agregar, modificar, consultar o eliminar 
        datos de una base de datos de SQL Server.

Las sentencias usadas son 4:

*Select:

Con este comando se extrae la data de las tablas.

Ejemplo:
select * from tabla

*Insert:

Con este comando se inserta datos en una tabla.

Ejemplo:
insert into tabla (campo1, campo2, campon) values(valor1, valor2, valorn)

*Update

Con este comando actualiza datos de una tabla.

Ejemplo:
Update tabla set campo1=valor1, campo2=valor2, campon=valorn

*Delete

Con este comando se podra eliminar datos de una tabla.

Ejemplo:
Delete from tabla where campo1=valor1
